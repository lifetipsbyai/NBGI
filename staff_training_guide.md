# Staff Training Guide - Noble Genius Institute Website

## Overview
This comprehensive training guide will help staff members effectively manage and update the Noble Genius Institute website using the WordPress content management system.

## Training Objectives
By the end of this training, staff will be able to:
- Navigate the WordPress dashboard confidently
- Create and edit content (pages, posts, events)
- Manage media files (images, documents)
- Update program information
- Post news and announcements
- Manage contact forms and inquiries
- Understand basic SEO principles
- Follow security best practices
- Perform basic troubleshooting

## Training Schedule

### Session 1: WordPress Basics (2 hours)
- WordPress dashboard overview

- Creating and editing pages

## Detailed Training Content

### 1. WordPress Dashboard Overview

#### 1.1 Logging In
- Access the login page: yourdomain.com/wp-admin
- Enter your username and password
- Two-factor authentication (if enabled)

#### 1.2 Dashboard Layout
- **Top Toolbar**: Quick access to common tasks
- **Main Menu**: Left sidebar navigation
- **Welcome Panel**: Quick links and recent activity
- **At a Glance**: Site statistics
- **Quick Draft**: Quick post creation
- **Activity**: Recent comments and posts

#### 1.3 User Roles
- **Administrator**: Full access to all features
- **Editor**: Can publish and manage posts/pages
- **Author**: Can publish and manage their own posts
- **Contributor**: Can write and manage their own posts but cannot publish
- **Subscriber**: Can only manage their profile

### 2. Content Management

#### 2.1 Creating Pages
1. Navigate to Pages → Add New
2. Enter page title
3. Add content using blocks
4. Set page attributes (template, parent page)
5. Add featured image
6. Preview and publish

#### 2.2 Creating Posts
1. Navigate to Posts → Add New
2. Enter post title
3. Add content using blocks
4. Select category and add tags
5. Add featured image
6. Set publish date/time
7. Preview and publish

#### 2.3 Using the Block Editor
- **Paragraph Block**: Basic text content
- **Heading Block**: Section headings (H1-H6)
- **Image Block**: Add images
- **Gallery Block**: Multiple images
- **List Block**: Bulleted or numbered lists
- **Quote Block**: Highlighted quotes
- **Button Block**: Call-to-action buttons
- **Columns Block**: Side-by-side content
- **Embed Block**: Embed videos and social media

#### 2.4 Content Best Practices
- Use clear, descriptive headings
- Keep paragraphs short and readable
- Use bullet points for lists
- Add alt text to all images
- Use internal linking
- Optimize content for SEO

### 3. Managing Programs

#### 3.1 Adding New Programs
1. Navigate to Programs → Add New
2. Enter program name as title
3. Add program description in content area
4. Set program type (ND, HND, Certificate)
5. Add program details in custom fields
6. Upload featured image
7. Publish program

#### 3.2 Program Custom Fields
- Program Type: ND/HND/Certificate
- Duration: 2 years/1 year/6 months
- Entry Requirements: Text area
- Career Prospects: Text area
- Tuition Fee: Number field
- Application Deadline: Date field

#### 3.3 Program Categories
- Computer Science
- Biotechnology
- Cybersecurity
- AI & Robotics
- Data Science
- Software Engineering
- Digital Marketing

### 4. Event Management

#### 4.1 Creating Events
1. Navigate to Events → Add New
2. Enter event title
3. Add event description
4. Set event date and time
5. Add event location
6. Set event category
7. Add registration link
8. Upload featured image
9. Publish event

#### 4.2 Event Categories
- Workshops
- Seminars
- Conferences
- Webinars
- Networking Events
- Academic Events
- Social Events

### 5. Media Management

#### 5.1 Uploading Media
1. Navigate to Media → Add New
2. Drag and drop files or click "Select Files"
3. Wait for upload to complete
4. Edit media details (title, alt text, caption)
5. Copy file URL if needed

#### 5.2 Media Library Organization
- Create folders for different content types
- Use descriptive file names
- Add alt text to all images
- Optimize images before uploading
- Regular cleanup of unused media

#### 5.3 Image Optimization
- Resize images before uploading
- Use appropriate file formats (JPEG for photos, PNG for graphics)
- Compress images to reduce file size
- Use responsive images

### 6. Form Management

#### 6.1 Contact Forms
1. Navigate to Contact → Contact Forms
2. Select the form to edit
3. Modify form fields as needed
4. Update form settings
5. Save changes

#### 6.2 Form Field Types
- Text fields
- Email fields
- Phone fields
- Dropdown menus
- Checkboxes
- Radio buttons
- Textareas
- File uploads

#### 6.3 Form Settings
- Email notifications
- Confirmation messages
- Spam protection
- Form styling
- Thank you page redirects

### 7. SEO Basics

#### 7.1 Yoast SEO Plugin
- Install and activate Yoast SEO
- Configure basic settings
- Add meta titles and descriptions
- Set up social media meta tags
- Configure XML sitemap

#### 7.2 On-Page SEO
- Use focus keywords
- Write meta descriptions
- Use heading tags properly
- Add alt text to images
- Create SEO-friendly URLs
- Use internal linking

#### 7.3 Content SEO
- Research keywords
- Write high-quality content
- Use keywords naturally
- Add related keywords
- Create comprehensive content

### 8. Security Best Practices

#### 8.1 Password Security
- Use strong, unique passwords
- Enable two-factor authentication
- Change passwords regularly
- Don't share passwords
- Use password managers

#### 8.2 User Management
- Only give necessary permissions
- Regular user access reviews
- Remove inactive users
- Monitor user activity
- Use strong usernames

#### 8.3 Content Security
- Regular backups
- Update plugins and themes
- Use security plugins
- Monitor for malware
- Secure file uploads

### 9. Troubleshooting

#### 9.1 Common Issues
- White screen of death
- Plugin conflicts

- Database errors
- Performance problems

#### 9.2 Basic Troubleshooting

6. Contact support

#### 9.3 Getting Help
- Check WordPress documentation
- Search support forums
- Contact plugin developers
- Ask in community forums
- Contact IT support

### 10. Performance Optimization

#### 10.1 Speed Optimization
- Use caching plugins
- Optimize images
- Minimize plugins
- Use CDN
- Optimize database

#### 10.2 Mobile Optimization
- Use responsive themes
- Test on mobile devices
- Optimize images for mobile
- Use mobile-friendly plugins
- Check mobile page speed
